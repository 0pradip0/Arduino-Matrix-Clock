# Arduino-Matrix-Clock

Simply Arduino Clock with LED matrix display 32 x 8 pix.

- Arduino Nano
- DS3231 module
- 4x LED matrix with MAX7219 driver
- 2x button

For use external power supply (no USB) you need
- 1N4148 diode
- 5V power supply
